# Proposal: 初始化核心 BSW 模块

> **变更ID**: init-core-bsw  
> **状态**: 活跃  
> **优先级**: P0  
> **负责人**: 予乐科技开发团队  
> **创建日期**: 2026-04-13

## 背景

予乐科技基础软件平台 (Yule BSW) 需要从零开始构建核心模块。作为 MVP 阶段的首要任务，我们需要建立基础架构并开发最核心的 BSW 模块。

## 目标

1. 建立 BSW 项目基础架构
2. 实现 MCAL 核心模块 (MCU, GPIO, CAN)
3. 建立代码生成器基础框架
4. 实现基础配置工具原型

## 范围

### 包含内容

- MCAL 层: MCU Driver, GPIO Driver, CAN Driver
- 配置工具: 基础配置界面原型
- 代码生成器: ARXML 解析 + C 代码生成
- 测试框架: 单元测试基础

### 不包含内容

- 完整 Service Layer 实现
- 高级诊断功能
- 云端同步功能
- 社区平台功能

## 关联 Specs

- `openspec/specs/bsw/spec.md` - BSW 整体规范
- `openspec/specs/toolchain/spec.md` - 工具链规范

## 验收标准

- [ ] MCAL 三个模块代码通过编译
- [ ] 代码生成器可生成基本配置代码
- [ ] 配置工具可展示基础界面
- [ ] 单元测试覆盖率 > 60%
