# 创建第一个项目

学习如何使用 YuleTech AutoSAR BSW Platform 创建第一个项目。

## 创建项目结构

```bash
./tools/yule-cli new-project MyFirstProject
```

## 项目目录

```
MyFirstProject/
├── configs/          # 配置文件
├── src/              # 应用代码
├── tests/            # 测试文件
└── build/            # 构建输出
```

## 配置系统

编辑 `configs/system.json` 来配置您的系统参数。
